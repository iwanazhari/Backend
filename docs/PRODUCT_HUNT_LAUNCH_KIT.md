## 🆘 **Crisis Management**

### **Common Issues & Solutions**

#### **Issue: Slow Start (< 50 votes by 9 AM)**
```
Solution:
1. Activate emergency contact list
2. Post in all Discord/Slack communities
3. DM close friends/colleagues
4. Consider paid promotion (Twitter ads)
```

#### **Issue: Negative Comments**
```
Solution:
1. Respond professionally and quickly
2. Acknowledge valid criticism
3. Offer to make it right
4. Don't argue publicly
5. Take complex issues offline

Example Response:
"Thanks for the feedback! You're right about [issue]. 
We're working on a fix. Can you DM me more details? 
Want to make this right."
```

#### **Issue: Technical Problems**
```
Solution:
1. Acknowledge immediately
2. Post status update on PH
3. Fix ASAP
4. Follow up when resolved
5. Offer compensation if needed

Example:
"Seeing some checkout issues. Team is on it! 
Use code SORRY20 for 20% extra off while we fix. 
ETA: 30 minutes."
```

#### **Issue: Competitor Attacks**
```
Solution:
1. Don't engage publicly
2. Report to Product Hunt if severe
3. Let community defend you
4. Focus on your supporters
5. Rise above it
```
