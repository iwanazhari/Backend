## 🚀 Quick Start (5 Minutes to First API)

```bash
# 1. Clone & Install
git clone <your-repo-url>
cd backend-starter-kit
npm install

# 2. Setup Environment
cp .env.example .env

# 3. Start Database (Docker)
docker-compose up -d postgres redis

# 4. Initialize
npm run prisma:generate
npm run prisma:migrate
npm run prisma:seed

# 5. Start Development
npm run dev

# Done! API running at http://localhost:3000
# Docs at http://localhost:3000/api/docs
```
