## 🆘 Troubleshooting

### **"No responses"**

```
Problem: <5% response rate

Solutions:
1. Your subject lines suck → A/B test 5 versions
2. Your message is too long → Cut by 50%
3. You're not personalizing → Research each person
4. Wrong audience → Try different persona
5. Wrong timing → Send at different times
```

### **"Demos but no sales"**

```
Problem: <20% demo-to-close rate

Solutions:
1. Not qualifying properly → Ask budget/timeline earlier
2. Demo too long → Cut to 20 minutes, focus on pain
3. Not handling objections → Use cheat sheet
4. No urgency → "Launch pricing ends in 48h"
5. Wrong pricing → Offer payment plan
```

### **"Can't get demos booked"**

```
Problem: People don't respond to calendar link

Solutions:
1. Send 2-3 time options instead of link
2. Do "exploratory call" not "demo"
3. Offer async video demo (Loom)
4. Build more trust before asking
5. Lower friction: "15 min chat" not "30 min demo"
```
