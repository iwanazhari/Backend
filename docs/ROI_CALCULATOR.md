## 🧮 **ROI Calculation Worksheet**

### **Step 1: Your Current Costs**

```
A. Developer Hourly Rate
   $ _____ /hour
   (If salaried: annual salary ÷ 2000 hours)
   (Typical: $50-150/hour)

B. Time Spent Per Project (Setup Only)
   _____ hours
   (Include: TypeScript config, folder structure, auth, 
    security, testing, Docker, docs, etc.)

C. Projects Per Year
   _____ projects/year
   (Freelancers: 4-8 typical)
   (Agencies: 10-20 typical)
   (Startups: 2-4 typical)

D. Annual Boilerplate Cost
   = A × B × C
   $ _____ /year
```

---

### **Step 2: With Backend Starter Kit**

```
E. Setup Time With Starter Kit
   _____ hours
   (Typical: 0.5-2 hours for first project)
   (Subsequent: 15-30 minutes)

F. Time Saved Per Project
   = B - E
   _____ hours/project

G. Annual Time Savings
   = F × C
   _____ hours/year

H. Annual Dollar Savings
   = G × A
   $ _____ /year
```

---

### **Step 3: Additional Benefits**

```
I. Faster Launch Value
   (Launching 2 weeks early = ? revenue)
   $ _____ /year

J. Reduced Bug Fix Time
   (Security done right = fewer emergencies)
   _____ hours × $ _____/hour
   $ _____ /year

K. Opportunity Cost Saved
   (What could you build with extra time?)
   (Side project? Another client? Learning?)
   $ _____ /year

L. Total Annual Benefits
   = H + I + J + K
   $ _____ /year
```

---

### **Step 4: Investment**

```
M. License Cost
   Basic:     $99
   Pro:       $199
   Enterprise: $499
   $ _____ (one-time)

N. Learning Time
   _____ hours × $ _____/hour
   $ _____ (one-time)

O. Total First-Year Investment
   = M + N
   $ _____
```

---

### **Step 5: ROI Calculation**

```
P. First-Year ROI
   = (L - O) ÷ O × 100
   _____ %

Q. Payback Period
   = O ÷ (L ÷ 12)
   _____ months

R. 3-Year Value
   = (L × 3) - O
   $ _____

S. 5-Year Value
   = (L × 5) - O
   $ _____
```

        function calculateROI() {
            // Get inputs
            const hourlyRate = parseFloat(document.getElementById('hourlyRate').value) || 0;
            const hoursPerProject = parseFloat(document.getElementById('hoursPerProject').value) || 0;
            const projectsPerYear = parseFloat(document.getElementById('projectsPerYear').value) || 0;
            const kitHours = parseFloat(document.getElementById('kitHours').value) || 0;
            const licenseCost = parseFloat(document.getElementById('licenseType').value) || 0;

            // Calculate current costs
            const currentAnnualCost = hourlyRate * hoursPerProject * projectsPerYear;

            // Calculate with starter kit
            const kitAnnualCost = hourlyRate * kitHours * projectsPerYear;

            // Time savings
            const hoursSavedPerProject = hoursPerProject - kitHours;
            const totalHoursSaved = hoursSavedPerProject * projectsPerYear;
            const annualSavings = currentAnnualCost - kitAnnualCost;

            // Investment
            const learningHours = 4;
            const learningCost = learningHours * hourlyRate;
            const totalInvestment = licenseCost + learningCost;

            // ROI
            const roi = ((annualSavings - totalInvestment) / totalInvestment) * 100;
            const paybackMonths = totalInvestment / (annualSavings / 12);
            const paybackDays = paybackMonths * 30;
            const threeYearValue = (annualSavings * 3) - totalInvestment;

            // Display results
            document.getElementById('roiValue').textContent = roi.toLocaleString('en-US', {
                maximumFractionDigits: 0
            }) + '%';
            document.getElementById('annualSavings').textContent = '$' + annualSavings.toLocaleString('en-US');
            document.getElementById('paybackPeriod').textContent = Math.round(paybackDays) + ' days';
            document.getElementById('threeYearValue').textContent = '$' + threeYearValue.toLocaleString('en-US');
            document.getElementById('roiResult').style.display = 'block';

            // Interpretation
            let interpretation = '';
            if (roi > 1000) {
                interpretation = `🎉 Incredible! You'll save ${Math.round(totalHoursSaved)} hours and $${annualSavings.toLocaleString()} per year. ` +
                    `That's like getting ${Math.round(annualSavings / hourlyRate)} hours of free time to build more, learn more, or spend with family. ` +
                    `Your investment pays for itself in just ${Math.round(paybackDays)} days!`;
            } else if (roi > 500) {
                interpretation = `✅ Great investment! You'll save $${annualSavings.toLocaleString()} per year, ` +
                    `with your investment paid back in ${Math.round(paybackDays)} days. ` +
                    `Plus, you'll ship projects faster and with better quality.`;
            } else if (roi > 100) {
                interpretation = `👍 Solid ROI. You'll save $${annualSavings.toLocaleString()} in your first year. ` +
                    `The real value might be in reduced stress and better code quality.`;
            } else {
                interpretation = `Consider the non-financial benefits: less stress, better code quality, ` +
                    `and more time for features that make your product unique.`;
            }
            document.getElementById('interpretation').textContent = interpretation;
        }
    </script>
